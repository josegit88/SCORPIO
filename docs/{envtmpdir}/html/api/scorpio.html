scorpio module¶

Este programa recibe información de RA y Dec de un par de galaxias interactuantes (o cercanas) u otros objetos individual y descarga los datos .fits correspondientes de datos del Sloan Digital sky Survey (SDSS) de una lista generada a partir de los filtros u, g, r, i, z. calcula la distancia en Mpc y la separación (solamente para el caso de pares).
